# Описание

**MultiLCB** — это универсальный live code bench с поддержкой множества языков программирования: Python, JavaScript, TypeScript, Go, Rust, Java, C#, C++, Ruby, PHP, Kotlin и Scala. Запускайте и тестируйте код в реальном времени в одном месте.
